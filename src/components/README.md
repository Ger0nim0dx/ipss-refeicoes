# 🍽️ IPSS Gestão de Refeições

Aplicação web desenvolvida em React para apoio à gestão de refeições em contexto IPSS.

O projeto foi desenvolvido com foco na organização da produção alimentar, controlo de custos, gestão de dietas especiais e visualização de dados administrativos.

---

# 🚀 Funcionalidades

## 📊 Dashboard Administrativo
- Total de refeições
- Custos diários e mensais
- Litros de sopa estimados
- Indicadores automáticos
- Visualização gráfica de dados

## 🥗 Gestão de Dietas e Alergias
- Registo de dietas especiais
- Gestão de alergias/intolerâncias
- Observações alimentares
- Organização por valências

## 📁 Histórico
- Registos diários
- Persistência local
- Consulta rápida de produção

## 📄 Exportação PDF
- Relatórios automáticos
- Relatórios de dietas
- Informação pronta para impressão

## 🌙 Interface Moderna
- Dark mode
- Layout responsivo
- Interface otimizada para desktop e mobile
